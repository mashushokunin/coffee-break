# Coffee Break - 設計書

## アプリの目的

ユーザーがコーヒーについて学び、新しいコーヒーを発見し、その時間をより豊かにするためのプラットフォーム。

## 主な機能

- [ ] コーヒー一覧表示
- [ ] コーヒー詳細情報表示
- [ ] ペアリング情報表示
- [ ] 音楽推薦
- [ ] お気に入り管理
- [ ] ローカルストレージでのデータ保存

## 画面構成

### 1. ホーム画面
- コーヒー一覧表示
- 検索・フィルター機能

### 2. コーヒー詳細画面
- コーヒー情報表示
- ペアリング情報
- 推奨音楽

### 3. お気に入り画面
- 保存したコーヒーの一覧

## データモデル

### Coffee（コーヒー）
```typescript
{
  id: string
  name: string
  origin: string
  roastLevel: string
  flavor: string[]
  description: string
  pairing: string[]
  musicGenre: string[]
}
```

## ユーザーフロー

1. ホーム画面でコーヒーを検索
2. コーヒーをクリックして詳細表示
3. ペアリング・音楽情報を確認
4. お気に入り保存
