# Coffee Break

コーヒーの魅力を発見・共有するWebアプリケーション

## 概要

Coffee Breakは、コーヒーを飲む時間をより楽しくするためのアプリです。コーヒーの情報、ペアリング、そして音楽のおすすめなどを提供します。

## 技術スタック

- **フロントエンド**: React + TypeScript
- **ビルドツール**: Vite
- **データストレージ**: JSON + localStorage

## インストール・起動方法

```bash
# 依存パッケージのインストール
cd frontend
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```
